# dungeon-pathfinding-cpp
Graph-based pathfinding with state-space search and combat simulation (C++)
## Description
This project implements a shortest path search in a dungeon with combat mechanics.

## Key Features
- State-space BFS (position + inventory)
- Combat simulation
- Inventory management (equip/drop)
- Path reconstruction

## Technologies
- C++
- STL (unordered_map, vector, deque)
